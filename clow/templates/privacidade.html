Politica de Privacidade

Ultima atualizacao: 04 de abril de 2026 — Conforme LGPD (Lei 13.709/2018)

1. Controlador dos dados

Daniel Baptista, Rio de Janeiro, RJ, Brasil.
Contato para questoes de dados: daniellbaptista2021@gmail.com

2. Dados coletados

Cadastro

Nome completo, email e senha. A senha e armazenada em formato hash (PBKDF2-SHA256) — nunca em texto puro.

Pagamento

Os dados de cartao de credito e PIX sao processados diretamente pelo Stripe. Nao armazenamos dados de pagamento nos nossos servidores.

API key (plano BYOK)

Se voce usa o plano gratuito, sua API key da Anthropic e armazenada no servidor do Clow para possibilitar as chamadas ao modelo.

Uso do servico

Historico de conversas com o agente, tokens consumidos, preferencias de uso, e configuracoes.

Dados tecnicos

Endereco IP, user agent do navegador e logs de acesso para seguranca e prevencao de abusos.

3. Finalidade do tratamento

  • Prestacao do servico: processar suas mensagens e retornar respostas do agente (base legal: execucao de contrato)
  • Billing: gerenciar assinaturas e faturamento (base legal: execucao de contrato)
  • Melhoria do servico: metricas agregadas e anonimizadas de uso (base legal: legitimo interesse)
  • Comunicacoes: notificacoes sobre o servico, franquia e atualizacoes (base legal: execucao de contrato)

4. Compartilhamento de dados

  • Anthropic: suas mensagens (prompts) sao enviadas para a API da Anthropic para processamento pelo modelo de IA. Sujeito aos termos da Anthropic.
  • Stripe: dados de pagamento processados pelo Stripe para cobranca. Sujeito aos termos do Stripe.
  • Nao vendemos, alugamos ou compartilhamos seus dados com terceiros para fins de marketing.

5. Armazenamento

Seus dados ficam em servidores no Brasil (VPS Hostinger, Sao Paulo). Servicos auxiliares como Supabase podem processar dados nos EUA, em conformidade com clausulas contratuais padrao.

6. Retencao

Seus dados sao mantidos enquanto sua conta estiver ativa. Apos a exclusao da conta, os dados sao removidos em ate 30 dias, exceto quando ha obrigacao legal de retencao (fiscal, por exemplo).

7. Seus direitos (LGPD Art. 18)

Voce tem direito a:

  • Confirmar se tratamos seus dados
  • Acessar seus dados pessoais
  • Corrigir dados incompletos ou incorretos
  • Solicitar anonimizacao ou eliminacao de dados desnecessarios
  • Portabilidade dos dados para outro servico
  • Saber com quem compartilhamos seus dados
  • Revogar seu consentimento a qualquer momento

Para exercer qualquer direito, envie email para daniellbaptista2021@gmail.com com o assunto "LGPD — [seu pedido]".

8. Cookies

Utilizamos apenas cookies de sessao para autenticacao (cookie "clow_session"). Nao usamos cookies de rastreamento ou de terceiros para publicidade.

9. Seguranca

  • Conexao HTTPS em todas as paginas
  • Senhas com hash PBKDF2-SHA256
  • Acesso restrito ao banco de dados
  • Logs de acesso monitorados
