Instalar o Clow no seu computador

Use o terminal completo do Clow na sua maquina. Crie sites, codigo, planilhas, documentos — tudo local, sem limite.

Requisitos

  • Windows 10 ou 11
  • Python 3.10 ou superior (baixar aqui)
  • Git (baixar aqui)
Como verificar se ja tenho Python e Git?

Abra o PowerShell (ou Prompt de Comando) e digite:

python --version

Deve mostrar algo como Python 3.12.x

git --version

Deve mostrar algo como git version 2.x.x

Se nao aparecer, instale pelos links acima. Na instalacao do Python, marque a opcao "Add Python to PATH".

Instalacao (3 minutos)

1

Abra o PowerShell

Clique com botao direito no menu Iniciar → "Terminal" ou "PowerShell"

2

Clone e instale o Clow

Copie e cole estes comandos um por um:

git clone https://github.com/daniellbaptista2021-lgtm/batmam.git clow
cd clow
python -m venv .venv
.venv\Scripts\activate
pip install -e .
3

Configure sua API Key

Plano pago — Instalacao automatica

Clique no botao abaixo pra gerar seu comando de instalacao personalizado. Tudo ja vem configurado — e so copiar e colar.

Copie e cole este comando no seu terminal:

⏰ Este comando expira em 24 horas 🔒 Uso unico — apos instalar, o comando e invalidado 🔑 Sua API key nunca e exposta
Plano gratuito (Traga sua Key):

Pegue sua key em console.anthropic.com e cole:

mkdir -p ~/.clow/app && echo "CLOW_PROVIDER=anthropic ANTHROPIC_API_KEY=sk-ant-SUA_KEY_AQUI CLOW_MODEL=claude-sonnet-4-20250514" > ~/.clow/app/.env
4

Rode o Clow

clow

Deve aparecer:

   ╔═══════════════════════════════════════╗
   ║   System Clow v0.2.0                   ║
   ╚═══════════════════════════════════════╝

✅

Pronto!

Agora voce pode usar o terminal completo do Clow. Exemplos:

clow "cria uma landing page pro meu restaurante"
clow "faz uma planilha de controle de estoque"
clow "cria uma API em Python pra integrar com WhatsApp"

Duvidas comuns

Qual a diferenca entre usar pela web e pelo terminal?

A web (clow.pvcorretor01.com.br) e pra configurar WhatsApp Agent, ver CRM Dashboard, gerenciar plano e settings. O terminal e pra criar coisas: landing pages, codigo, planilhas, documentos. O terminal roda na sua maquina, nao sobrecarrega nada.

Preciso instalar pra usar o WhatsApp Agent?

Nao. O WhatsApp Agent e configurado pela web, sem instalar nada. A instalacao no PC e so pra quem quer usar o terminal pra criar coisas.

Posso usar nos dois? Web e terminal?

Sim. Usa a web pra WhatsApp e CRM, e o terminal pra criacao. Sao complementares.

O terminal gasta da minha franquia?

Sim, usa tokens da mesma franquia do seu plano. No plano gratuito, usa sua propria API key.

Deu erro na instalacao, o que faco?

Os erros mais comuns: Python nao instalado (instale pelo python.org), Git nao instalado (instale pelo git-scm.com), ou PATH nao configurado (reinicie o terminal apos instalar). Se persistir, mande o erro no suporte.
