# Cloned Site

Generated by Clow Website Cloner. Next.js 15 + Tailwind v3 + shadcn/ui.

```bash
npm install
npm run dev    # http://localhost:3000
npm run build  # production
```

Contents:
- `src/app/` — App Router (layout, page, globals.css)
- `src/components/` — generated React components per section
- `docs/research/` — recon data, behaviors, per-component specs
- `public/` — downloaded images, icons, videos

Original URL is in `docs/research/RECON.json`.
