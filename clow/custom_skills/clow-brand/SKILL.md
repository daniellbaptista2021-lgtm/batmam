# Clow Brand Guidelines

## Visual Standards

### Color Palette
- Primary: #8B5CF6 (Roxo Clow)
- Primary Dark: #6D28D9
- Primary Light: #A78BFA
- Background Dark: #0A0A15
- Background Medium: #111118
- Text Primary: #FFFFFF
- Text Secondary: #E0E0E0
- Text Muted: #9D9DB5
- Success: #10B981 (Verde)
- Warning: #F59E0B (Amarelo)
- Error: #EF4444 (Vermelho)

### Typography
- Font Family: Inter, system-ui, sans-serif
- H1: 28pt Bold, color #8B5CF6
- H2: 22pt Semibold, color #FFFFFF
- H3: 16pt Semibold, color #A78BFA
- Body: 11pt Regular, color #E0E0E0
- Small: 9pt Regular, color #9D9DB5

### Document Standards
- Header: Logo "Clow" em roxo, alinhado a esquerda
- Footer: "Gerado por Clow AI" centralizado, cor #9D9DB5
- Margens: 2.5cm todas
- Graficos: usar paleta roxa como cor principal

### Spreadsheet Standards
- Header row: background #8B5CF6, texto branco, bold
- Alternating rows: #F5F3FF e #FFFFFF
- Borders: #E0E0E0 fina
- Currency: formato R$ brasileiro
- Dates: formato DD/MM/YYYY

### Presentation Standards
- Slide background: gradiente #0A0A15 para #111118
- Title slides: texto grande em #8B5CF6
- Content slides: fundo escuro, texto branco
- Accent shapes: roxo #8B5CF6
- Charts: paleta roxa (#8B5CF6, #6D28D9, #A78BFA, #C4B5FD)
